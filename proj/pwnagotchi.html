Pwnagotchi

You can do a ton of stuff with microcomputers. In cybersecurity, you will end up with an endless list of projects. All you need is the luxury of having the time to dive into most of them. After gaining some hands-on experience with Raspberry Pi Zero 2W (02w) during one of my internships, I have decided to configure my own. Out of the things you can do with a 02w, I went with setting up my own pwnagotchi.

What is a Raspberry Pi?

A Raspberry Pi is a low-cost, small/compact, single-board computer (SBC) produced by the Raspberry Pi Foundation. They were mainly established to promote computing education. Essentially, they function as a complete computer (without the peripherals) and widely used for DIY projects. Usually (depending on the model), all you need in hand to operate it is a data-transfer cable (yes, the one with the USB trident symbol) and a micro-SD card.


Pwnagotchi

This little guy is mainly used to capture WPA key frames (4-way handshake) but can be used to monitor and inspect the network environment. It uses bettercap (which is an open-source program designed for network recon, MITM, packet sniffing, etc) and the firmware is flashed on an SD card and gets inserted into most SBCs. I prefer using the Pi Zero due to the compact build it has. You don't really need much. I added in an e-ink display since it comes with an interactive (kinda) UI. It's adorable.

Why capture WPA key frames? I wrote about how 4-way handshakes work but simply put, we need those key frames to bruteforce our way into gaining the passphrase. We can already do this with certain tools like the aircrack-ng suite. Pwnagotchi just gives the capturing and cracking process a gamified twist.


Features

Pwnagotchi has its own name and level status, along with preset text bubbles when something happens. Experience points are gained when it detects a network and when it successfully captures its key frames. Pwnagotchi also comes with expressive emotes and it changes based on its environment, like when it detects a nearby network. Automatically, pwnagotchi will attempt to capture a network's key frames by doing deauthentication and association attacks. Captures, along with its config files, are stored within pwnagotchi and can be accessed via ssh (and a DATA-TRANSFER cable [yes i'll say it again]). I believe they can also crack passwords but I haven't gone around and figured that out yet. Conventionally, you can retrieve the captures from pwnagotchi and use programs like aircrack to rock it.

Since pwnagotchi is a community project, it is open-source and is open for further development. Newer addition to the device includes a web UI and AI mode. The web UI gives access to add certain plugins, such as age, memory and temperature trackers (personal favourite).

How are they vulnerable? How can we exploit it? How can we make them more secure? These are the questions that you would ask yourself when tinkering with gadgets like this.

I believe that hands-on experience is crucial when learning about how systems work. Once I have understood the basics, I'd like to use physical tools to test my understanding and of course, to have a little fun. It gets you doing things and think, repeat it and then it sticks with you.
